# Hướng dẫn sử dụng git

## Tạo nhánh mới trên repo

<ol>
    <li>git clone ${remote url} </li>
    <li>git checkout -b ${tên nhánh}</li>
    <li>git push -u origin ${tên nhánh}</li>
</ol>

---

## Push code 

<ol>
    <li>git add .</li>
    <li>git commit -m "Mô tả về commit"</li>
    <li>git push</li>
</ol>
